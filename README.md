# قدس — ثيم الموقع الإخباري

ثيم الصفحة الرئيسية لموقع **شبكة قدس الإخبارية** (واجهة عربية RTL).

## بنية المشروع

```
quds-ar/
├── index.html                 # الصفحة الرئيسية (نقطة الدخول)
├── assets/
│   ├── js/
│   │   └── support.js         # محرّك التشغيل (DC runtime) — يفسّر القالب
│   └── img/
│       ├── logo.png           # شعار الشبكة
│       └── telegram-banner.png# بانر قناة تيليغرام
├── docs/
│   └── reference/             # مواد مرجعية للتصميم (ليست جزءًا من الثيم)
│       ├── original-site-screenshot.jpg
│       ├── telegram-banner-source.png
│       └── design-thumbnail.webp
├── .gitattributes
├── .gitignore
└── README.md
```

## التشغيل

الصفحة تعتمد على محرّك `support.js` الذي يحمّل **React** و**Babel** من شبكة CDN
(`unpkg.com`)، لذا يلزم **اتصال إنترنت** لعرضها، ويُفضّل تشغيلها عبر خادم محلّي
بدل فتح الملف مباشرة:

```bash
# أي خادم ثابت يكفي، مثلاً:
python -m http.server 8000
# ثم افتح: http://localhost:8000/
```

## محتوى الصفحة

كل بيانات الأخبار والأقسام والروابط معرَّفة داخل بلوك
`<script type="text/x-dc">` في نهاية ملف `index.html` (الدالة `renderVals`)،
وهي حاليًا بيانات تجريبية تشير إلى `qudsn.co`. عدّلها هناك لتغيير المحتوى.

## التخصيص

يدعم الثيم عبر خصائص المكوّن:

| الخاصية        | النوع   | الافتراضي  | الوصف                     |
|----------------|---------|------------|---------------------------|
| `accentColor`  | لون     | `#2f7fcf`  | اللون الرئيسي للموقع       |
| `cornerRadius` | رقم     | `12`       | درجة تدوير الحواف (px)     |
| `showTopBar`   | منطقي   | `true`     | إظهار الشريط العلوي        |
