# قدس — ثيم الموقع الإخباري (متعدّد الصفحات)

ثيم متكامل لموقع **شبكة قدس الإخبارية** — واجهة عربية (RTL) جاهزة للإنتاج،
مبنية بـ **HTML5 + CSS3 + Vanilla JavaScript** بدون أي إطار عمل (لا React/Vue/Angular/Tailwind).

## الصفحات

| الملف | الوصف |
|------|-------|
| `index.html` | الصفحة الرئيسية |
| `about.html` | من نحن (تعريف، رسالة ورؤية، مبادئ، أرقام، منصّات، CTA) |
| `contact.html` | اتصل بنا (معلومات + نموذج + خريطة رام الله) |
| `articles.html` | الأرشيف (فلترة حسب القسم + Pagination) |
| `category.html` | صفحة القسم (العنوان من `?cat=`) |
| `tag.html` | صفحة الوسم (العنوان من `?tag=`) |
| `single-article.html` | الخبر المفرد (Header + جسم المقال + مشاركة + Sidebar + ذات صلة + سابق/تالي) |
| `author.html` | صفحة الكاتب (نبذة + إحصاءات + مقالاته) |
| `videos.html` | الفيديو (Hero + شبكة كروت فيديو + Swiper) |
| `podcasts.html` | البودكاست (شبكة حلقات + Swiper) |
| `single-podcast.html` | حلقة بودكاست (غلاف + مشغّل صوت + وصف + مشاركة + حلقات مشابهة) |
| `search-results.html` | نتائج البحث (شريط بحث + عدّاد نتائج + Empty State) |
| `404.html` | صفحة الخطأ (رسالة + أزرار + أخبار مقترحة) |

### المعمارية المشتركة (منتج واحد)
- **`components.js`** يحقن الشريط العلوي + الهيدر + التنقّل + الفوتر + نافذة البحث في
  العناصر `#qd-header` / `#qd-footer` / `#qd-search` بنفس كلاسات الصفحة الرئيسية، مع
  تفعيل القسم النشِط عبر `body[data-page]`. مصدر واحد للهيدر/الفوتر = اتساق تام.
- **`main.js`** يهيّئ تلقائيًا أي `.quds-swiper` (حسب `data-swiper`)، AOS، العدّادات،
  أزرار المشاركة (`[data-share]`)، مشغّل الصوت (`.audio-player`)، نموذج التواصل،
  الفلاتر (`.filter-chip`)، وعناوين من باراميتر الرابط (`data-from-query`).

## المكتبات (CDN)
Swiper.js 11 · AOS 2.3 · Font Awesome 6.5 · خط IBM Plex Sans Arabic.
> تحتاج الصفحات اتصال إنترنت لتحميل هذه المكتبات وصور الأخبار من `qudsn.co`.


## فحوص الجودة المنجَزة
- ✅ صفر overflow أفقي على 320 / 375 / 480 / 768 / 992 / 1200 / 1440 لكل الصفحات
- ✅ هيدر/فوتر/بحث موحّد ومحقون عبر كل الصفحات (تنقّل داخلي متكامل)
- ✅ Swiper للأقسام الكثيفة · مشغّل صوت وظيفي · فلترة وبحث وEmpty State · عدّادات
- ✅ مشاركة اجتماعية (فيسبوك/إكس/واتساب/تيليغرام/نسخ الرابط) في صفحات المحتوى
- ✅ Lazy-loading + `alt` لكل صورة + `aria-label` + HTML دلالي + `prefers-reduced-motion`
- ✅ صفر أخطاء في الـ console
